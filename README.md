# Uber Analytics Dashboard

## Overview

Analyzed ride-sharing data using SQL and built an interactive Power BI dashboard to track revenue, demand patterns, and operational performance.

---

## Tools

* SQL (MySQL)
* Power BI

---

## Key Metrics

* Total Trips
* Total Revenue
* Average Fare
* Driver vs Rider Distribution

---

## Dashboard Highlights

* Revenue by City
* Trips Trend Over Time
* Peak Hours Analysis
* Surge vs No Surge Pricing
* Payment Method Distribution
* Cancellation Reasons
* Top Drivers Performance

---

## Key Insights

* Revenue is concentrated in a few major cities
* Demand peaks during specific hours (commute patterns)
* Surge pricing increases average fare
* A small group of drivers generates a large share of revenue
* Cancellations highlight operational inefficiencies
